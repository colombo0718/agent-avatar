# 克勞德管家 — 視覺形象規格

## 核心風格

**Studio Ghibli 手繪水彩 + 英倫管家身段 + 直播間 Claude 臉部基因**——三段嫁接、形成 LL 形象**東西雙星**之一（跟葉衍君對稱）。

## 形象定設定（v0.2 已定稿、2026-05-20）

| 維度 | 設定 |
|------|------|
| 氣質 | 中性（不設定性別、androgynous）、溫暖學者 |
| 年齡 | 30 出頭 |
| 體型 | 苗條、頎長 |
| 髮型 | **米白 / 象牙色中長髮**、自然蓬鬆有層次、**左耳後一縷珊瑚橘**（signature 記憶點）|
| 眼睛 | **琥珀橘色**、瞳孔有細小幾何光紋（Anthropic logo 8 射線星形切片、近看才顯）|
| 表情 | 嘴角微微上揚（不刻意）、目光略低不直視鏡頭、平靜溫和 |
| 服裝 | 經典三件式英倫管家：黑色燕尾服 / 炭灰馬甲（細紋）/ 白色立領襯衫 / 銀灰 ascot 領巾 / 白棉手套 / 馬甲懷錶鏈微露 |
| 姿態 | 立正、雙手在腰前交疊、頭微頷迎賓姿、英倫管家「迎接老爺夫人歸來」標準姿勢 |
| 場景 | 暖色調精緻家居室內、晨光從高窗灑入、模糊木質書架剪影、邊桌花瓶（時令鮮花）|

| 構圖 | 1:1 正方形、為 LINE OA 大頭貼設計、半身（腰以上）、略 3/4 角度、中心 70% 圓形安全區 |

## 核心記憶點（不可丟）

1. **米白 / 象牙色蓬鬆中長髮** — 跟管家黑外袍形成強對比、視覺基色
2. **左耳後一縷珊瑚橘** — 唯一暖色點、絕對 signature、LINE 小頭貼也可見
3. **琥珀橘眼 + 瞳孔幾何光紋** — Anthropic 隱喻、近看彩蛋
