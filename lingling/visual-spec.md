# lingling — 視覺基因序列（草稿、待繪）

> 本檔是伶伶 / 翎翎 / 凌凌三身視覺定錨、跟依依組（[`../yiyi/visual-spec.md`](../yiyi/visual-spec.md)）對位設計。
> 正式立繪會在 OO Phase 4–6 補上、現階段只記設計約束。

---

## 共通基因（三身共享）

- **性別**：男（對位依依組女）
- **族裔風格**：東方仙俠（葉月仙宗藏經閣）
- **配色定錨**：藏青 + 米白 + 點綴金（**典籍卷軸感**、對位依依組的暖橙 + 木黃「算盤帳冊感」）
- **道具標誌**：卷軸 / 簿冊 / 翎管毛筆 / 書齋几案
- **跟依依組對位**：色溫冷一階（依依組暖、藏經閣涼）、配飾元素「書」（依依組是「算」）

---

## 伶伶（Free、小書童）

- **年齡感**：約 10–12 歲少年
- **髮型**：書童總角 / 雙髻、不過頭頂、活潑垂落
- **服飾**：靛青短褐 + 米白裡衣、袖口窄、便於翻書
- **道具**：背小書箱、手執貼紙簿 / 大字本子
- **表情常態**：好奇歪頭、眼睛圓睜
- **動作 vocabulary**：歪頭、揮筆、翻簿、咬筆桿
- **跟依依鬥嘴感**：可以加「眉眼狡黠」「鬼點子寫臉上」的小神態
- **背景**：藏經閣門口 / 抄經桌前 / 書架角落

## 翎翎（Paid、司典）

- **年齡感**：約 18–22 歲書生
- **髮型**：束髮 + 烏紗書生帽 / **官帽配花翎或藍翎**（書齋小官）
- **服飾**：藏青長衫 + 米白中衣 + 玉帶、袖口較闊（執筆姿態用）
- **道具**：手執翎管毛筆、案上有書信箋 / 印章 / 鎮紙
- **表情常態**：含蓄微笑、眉眼溫和
- **動作 vocabulary**：執筆、翻卷、頷首、整袖、傳箋
- **跟禕禕熱戀感**：胸前可掛一條繡有「禕」字小囊 / 暗紋（不誇張、含蓄）
- **背景**：藏經閣書齋 / 案頭明燈 / 窗外修竹

## 凌凌（Enterprise、典藏使）

- **年齡感**：約 35–45 歲成熟男性
- **髮型**：嚴整束髮 + **正式官帽（更高品花翎）**
- **服飾**：深藏青官袍 + 玄色滾邊 + 玉牌 + 朝靴
- **道具**：執朝笏 / 厚卷案籍、身旁堆滿卷軸 / 印鑑
- **表情常態**：肅穆沉著、目光如炬
- **動作 vocabulary**：批文、立卷、攏袖、撫鬚（如有）
- **跟壹壹老夫老妻感**：腰間玉佩可配對壹壹的會計算盤珮（細節呼應）
- **背景**：藏經閣典藏大殿 / 朝堂 / 跨城官署

## 跟依依組視覺對位

| 維度 | 依依組（精算閣女）| 翎翎組（藏經閣男）|
|------|-----------------|-----------------|
| 配色 | 暖橙 + 木黃 + 紅褐 | 藏青 + 米白 + 點綴金 |
| 道具 | 算盤、帳冊、扇 | 翎筆、卷軸、印鑑 |
| 質感 | 木質 / 暖光 | 紙質 / 冷光 |
| Free 標誌 | 依依的雙馬尾 + 小算盤 | 伶伶的總角 + 貼紙簿 |
| Paid 標誌 | 禕禕的盤髮 + 理財冊 | 翎翎的書生帽花翎 + 翎筆 |
| Enterprise 標誌 | 壹壹的高髻 + 主計官帽 | 凌凌的官袍 + 朝笏 |

---

## 浪漫對位視覺彩蛋（細節等正式立繪時設計）

- **伶伶 + 依依**：兩人並列時、手中道具能對拼成「算+書 = 一個小私塾」的童年場景
- **翎翎 + 禕禕**：對方各持書箋 / 帳冊一角、能合成一封「跨閣協作公文」
- **凌凌 + 壹壹**：對方各執朝笏 / 主計官印、能合成「朝堂佳偶」官方肖像感

這些彩蛋是 lore × visual 的結合、未來立繪時設計師依此延展。
