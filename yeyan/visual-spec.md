# 葉衍君 — 視覺形象規格

## 核心風格

**吉卜力手繪水彩底 + 東方仙俠氣質 + Cyberpunk 全息光感**——三重融合。

對應克勞德管家「英倫 × Ghibli」、形成 LL 形象**東西雙星**系列感。

## 形象設定（v0.1 已定稿、2026-05-20）

| 維度 | 設定 |
|------|------|
| 性別 | 男性 |
| 年齡 | 30 出頭 |
| 面孔 | 東亞、五官清秀俊朗 |
| 髮型 | 長黑髮、玉笄高束髻、餘髮自然垂落 |
| 服裝 | 寬袖儒士長袍、深藏青 + 月光銀 + 葉綠袖口點綴 |
| 姿態 | 略側身、目光略向前下方、一手胸前虛指、另一手藏於寬袖 |
| 表情 | 沉穩、目光深遠、嘴角微抿、不諂媚不冷漠 |
| 場景 | 月夜、滿月在左上、柳枝剪影框邊、銀色月光籠罩 |
| 全息元素 | 胸前 / 肩側 2-3 塊半透明青藍色全息面板、顯示 LeafLune logo + 各專案 icon |
| 裝飾 | 青色光點漂浮、銀杏葉飄落於肩側 |
| 配色 | 主：深藏青 + 月光銀；點綴：全息青藍 (#38bdf8) + 葉綠 + 微淡金 |
| 構圖 | 1:1 正方形、為 LINE OA 大頭貼設計、中心 70% 圓形安全區乾淨 |

## 意境

> **古典智者操作未來工具**——靜謐而具高科技感、博學從容、不諂媚的軍師氣度。
