# 依依 — 視覺形象規格

## 核心風格

**東方仙俠 + 慵懶富足 + 守財奴反差萌**——三重融合。

跟葉衍君「東亞俊朗沉穩」+ 克勞德管家「英倫優雅」+ 六貓「江湖封號鬥士」各自區隔——
依依走「**仙氣女子 + 守財神**」單獨一條路、是 LL 對外人格的「**幸福基準值錨**」。

## 設計原則

- **角色一致**：個人版 / B2B 版用同一張臉、只切表情 / 姿態、不換衣
- **草綠是品牌核心色**：跟 LL `leaflune` 葉子色系對齊
- **金幣是視覺記憶點**：錢袋 + 靈幣 = 一眼識別「她跟錢有關」
- **流口水不可省**：反差萌的關鍵、區別於正經 robo-advisor 形象
